Architecture

Core Systems

Eight integrated subsystems built entirely from open-source components. Each module is independently upgradeable and field-replaceable.

>_

Powertrain

VESC 6.9 QS Hub Motors LiFePO4

Open-source VESC motor controllers with Field-Oriented Control driving QS 5kW BLDC hub motors. Three-level regenerative braking recovers 10-33% energy. 48V LiFePO4 modules with 2,000-5,000 cycle life, hot-swappable in under 2 minutes via Anderson SB350 connectors.

~*~

Energy Harvesting

Solar 1.2kW Regen Braking Regen Suspension

Triple energy recovery: 400W integrated + 800W deployable solar panels, VESC regenerative braking, and electromagnetic regenerative suspension harvesting 100-300W on rough terrain. Combined solar output: 1,800-3,500 Wh/day in good conditions.

[=]

Ruggedization

MIL-STD-810H IP67 EMP Shielded

Designed to MIL-STD-810H for vibration, shock, sand, dust, and extreme temperature (-32C to +49C). IP67/IP68 sealed electronics, Faraday cage enclosures on all critical systems, multi-layer surge protection, and shielded twisted-pair cabling throughout.

~O~

Water Systems

Sawyer 0.1um UV-C Rainwater

On-board water independence: Sawyer Squeeze 0.1 micron filtration (99.99999% bacteria removal), UV-C purification for viruses, 12V diaphragm pump at 3.5 GPM, 50-gallon collapsible bladder storage, and rainwater collection integrated into the solar canopy.

((.))

Communications

Meshtastic ATAK Starlink Iridium

Multi-layer encrypted comms: Meshtastic LoRa mesh (AES-256, 10+ km per node, infinite relay), ATAK tactical awareness, goTenna Pro X2 (100+ mi aerial relay), Starlink Mini broadband, and Iridium GO! exec for true global satellite coverage.

{0}

Digital Sovereignty

Raspberry Pi 5 Nextcloud AES-256

Self-hosted digital infrastructure: Raspberry Pi 5 running Nextcloud, Vaultwarden, WireGuard VPN, Pi-hole, and Grafana dashboards. 1TB encrypted NVMe storage (LUKS + VeraCrypt). Offline maps, offline crypto wallets. No cloud dependency. You own all data.

/|\

Shelter & Tools

Deploy Shelter 3D Printer Earthbag

Field-deployable shelter system: tarp, hammock, bivy, and pop-up canopy with solar mount points. Creality Ender 3 for printing replacement parts in the field. Battery-powered stick welder and earthbag construction tools for semi-permanent structures.

.*.

Regenerative Agriculture

Seed Dispersal Soil Sensing Biochar

Leave the land better: ATV-mount broadcast seeder for cover crops, on-the-go soil EC sensor with RTK GPS field mapping, and a mobile TLUD biochar micro-kiln producing carbon-negative soil amendment. Inoculant system for activated biochar application.

Build Tiers

Start with a rideable electric quad for $5K. Scale up to a full sovereign platform. Every tier is community-buildable in a makerspace.

Tier 1
$5K - $8K

The Starter

A rideable electric quad with basic sovereignty features. The entry point for makers and homesteaders.

  • Single 5kW QS hub motor, 25 mph
  • VESC 6.9 controller with regen braking
  • 1x 48V/100Ah LiFePO4 module (4.8 kWh)
  • 25-40 mile road range
  • 100W flexible solar panel
  • 2x Meshtastic mesh nodes
  • Raspberry Pi Zero 2W + encrypted storage
  • Sawyer Squeeze water filter
  • Chromoly tube frame, basic suspension
  • Basic tarp shelter
Build time: 2-4 weeks Skill level: Intermediate Tools: MIG welder, hand tools, soldering iron
Tier 2
$12K - $18K

The Homesteader

Full sovereignty stack with regenerative capabilities. Energy, water, comms, and digital independence.

  • 2x 5kW hub motors, AWD, 40 mph
  • 2x VESC 6.9 + three-level regen braking
  • 2x 48V modules (9.6 kWh), hot-swappable
  • 50-80 mile road range
  • 300W integrated + 400W deployable solar, MPPT
  • V2H power export: 3 kW bidirectional
  • Meshtastic + ATAK tablet + Garmin inReach
  • Raspberry Pi 5 + Nextcloud sovereignty stack
  • Sawyer + UV purifier + pump + 20-gal bladder
  • Hemp composite body, UHMWPE skid plates
  • Broadcast seeder + soil moisture sensor
  • Faraday enclosures on critical electronics
Build time: 4-8 weeks Skill level: Advanced Tools: TIG/MIG welder, composite layup, electronics bench
Tier 3
$25K - $35K

The Expeditionary

Military-grade sovereign platform with full regenerative integration. Maximum capability and resilience.

  • 4x 5kW hub motors, true 4WD, torque vectoring
  • 4x VESC + Axiom backup, 55 mph
  • 4x 48V modules (19.2 kWh), field-swappable
  • 80-140 mile road range
  • 1,200W solar (400W integrated + 800W deployable)
  • V2H: 5-10 kW bidirectional inverter
  • Meshtastic + ATAK + Starlink + Iridium + goTenna + Ham
  • Full digital sovereignty stack + backup SBC
  • Complete water system + 50-gal + rainwater capture
  • Full EMP hardening: Faraday + SPDs + shielded cabling
  • Regen electromagnetic suspension (100-300W)
  • Biochar micro-kiln + RTK GPS soil mapping
  • 3D printer + stick welder + earthbag tools
  • FLIR thermal for night navigation
Build time: 2-6 months Skill level: Expert Tools: Full fab shop, electronics lab, CNC helpful
