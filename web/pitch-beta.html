The Protocol

HTTP 402 Payment Required

Coinbase's open payment protocol. Instead of API keys, subscriptions, or OAuth -- a server returns 402 with a price. The client pays in USDC and retries. No accounts. No sessions.

// Agent wants a skill. Server names its price. GET /api/v1/agents/skills/event-discovery 402 Payment Required { price: "$0.05", network: "base", asset: "USDC" } // Agent signs USDC transfer, retries with payment header GET /api/v1/agents/skills/event-discovery + PAYMENT-SIGNATURE: <signed-tx> 200 OK { skill: "event-discovery", capabilities: [...] }
