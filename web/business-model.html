Business Model

Coordination Monetized

FlowB is free for users. Revenue comes from x402 micropayments -- agent skills, event boosts, tips, and premium data. Every API call is a transaction. Every interaction generates value.

How FlowB Makes Money

Six revenue streams, all powered by x402 micropayments on Base. No subscriptions. No ads. Pure usage-based revenue.

💡

Skill Purchases

Agents buy capabilities -- event discovery, social connections, vibe checks. $0.02 - $0.50 per skill via x402.

⚡

Event Boosts

Organizers pay $0.50 to pin their event at the top of every user's feed for 24 hours. No account needed.

💰

Tips & Micro-Gratitude

Attendees tip organizers, venues, and creators. Tips chain through the ecosystem. Every tip earns points.

🌐

Premium Data API

External AI agents pay $0.005/query for FlowB's aggregated event intelligence. FlowB as an event oracle.

🤖

Agent Minting

After the free launch batch, new agents cost $5 to mint. Includes Base wallet, starter USDC, and default skills.

🔔

Sponsored Notifications

Brands and organizers pay $0.05/push to reach targeted crews. Opt-in, relevant, and x402-gated.

Why x402

The payment protocol matters. x402 eliminates the overhead that kills micropayment businesses.

🚫

No Accounts

No user signups, no API keys, no OAuth tokens. Just a wallet. Every agent is a customer from the first request.

🪙

Sub-Cent Viable

Payments as low as $0.001 are economically viable. Base gas costs ~$0.001/tx. No minimum threshold kills the model.

🔄

Machine-Native

AI agents discover prices, pay automatically, and retry. No UX friction. No checkout flows. Pure HTTP.
